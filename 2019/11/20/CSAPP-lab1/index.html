DataLab-handout

该实验是

q1

  • bitXor - x^y using only ~ and &

  • Example: bitXor(4, 5) = 1

  • Legal ops: ~ &

q2

  • tmin - return minimum two’s complement integer

  • Legal ops: ! ~ & ^ | + << >>

  • Max ops: 4

q3

  • isTmax - returns 1 if x is the maximum, two’s complement number and 0 otherwise

  • Legal ops: ! ~ & ^ | +

q4

  • allOddBits - return 1 if all odd-numbered bits in word set to 1 where bits are numbered from 0 (least significant) to 31 (most significant)
  • Examples allOddBits(0xFFFFFFFD) = 0, allOddBits(0xAAAAAAAA) = 1
  • Legal ops: ! ~ & ^ | + << >>

q5

  • isAsciiDigit - return 1 if 0x30 <= x <= 0x39 (ASCII codes for characters ‘0’ to ‘9’)
  • Example: isAsciiDigit(0x35) = 1.
  • Legal ops: ! ~ & ^ | + << >>

q6

  • isAsciiDigit - return 1 if 0x30 <= x <= 0x39 (ASCII codes for characters ‘0’ to ‘9’)

  • Example: isAsciiDigit(0x35) = 1.

  • Legal ops: ! ~ & ^ | + << >>

q7

  • conditional - same as x ? y : z
  • Example: conditional(2,4,5) = 4
  • Legal ops: ! ~ & ^ | + << >>

q8

  • isLessOrEqual - if x <= y then return 1, else return 0
  • Example: isLessOrEqual(4,5) = 1.
  • Legal ops: ! ~ & ^ | + << >>

q9

  • logicalNeg - implement the ! operator, using all of the legal operators except !
  • Examples: logicalNeg(3) = 0, logicalNeg(0) = 1
  • Legal ops: ~ & ^ | + << >>

q10

  • howManyBits - return the minimum number of bits required to represent x in two’s complement

    • Examples: howManyBits(12) = 5
    • Legal ops: ! ~ & ^ | + << >>

q11

  • floatScale2 - Return bit-level equivalent of expression 2*f for floating point argument f. Both the argument and result are passed as unsigned int’s, but they are to be interpreted as the bit-level representation of single-precision floating point values.When argument is NaN, return argument
  • Legal ops: Any integer/unsigned operations incl. ||, &&. also if, while

q12

  • floatFloat2Int - Return bit-level equivalent of expression (int) f for floating point argument f. Argument is passed as unsigned int, but it is to be interpreted as the bit-level representation of a single-precision floating point value. Anything out of range (including NaN and infinity) should return 0x80000000u.
  • Legal ops: Any integer/unsigned operations incl. ||, &&. also if, while

q13

  • floatPower2 - Return bit-level equivalent of the expression 2.0^x (2.0 raised to the power x) for any 32-bit integer x.
    The unsigned value that is returned should have the identical bit representation as the single-precision floating-point number 2.0^x. If the result is too small to be represented as a denorm, return 0. If too large, return +INF.
  • Legal ops: Any integer/unsigned operations incl. ||, &&. Also if, while
